HTML Review

What does HTML stand for?
Hypertext Markup Language

What is the difference between <head> and <body>?
Head manages the metadata read by the browser, Body manages the content displayed

Name 3 semantic HTML elements
<header></header>
<main></main>
<footer></footer>
<nav></nav>
<article></article>
<aside></aside>
<section></section>

What attribute does an <a> tag need to create a link?
<a href="https://www.google.com">

What is a self-closing HTML tag?
Self closing tags are tags that only need one opening tag to function.
<br>, <img>, <meta>, <link>

What does <!DOCTYPE html> do?
This tells the browser that the file is a HTML file.

Every HTML file has the same skeleton structure
